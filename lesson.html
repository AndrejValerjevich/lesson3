flex-direction

order

Спасите котов от псин с помощью order (коты должны быть вместе с котами, псины вместе с псинами)

flex-wrap

Спасите псин от сжатия с помощью flex-wrap (размер собаки должен стать изначальным)

justify-content

Отцентруйте котяру по горизонтали с помощью justify-content

align-items

Отцентруйте котяру по вертикали с помощью align-items

Отцентруйте псину по центру контейнера с помощью align-items и justify-content
